1. Залить клиентскую часть из папки Client на сервер.
2. Залить серверную часть из папки Server на сервер.
3. Залить базу данных pilot.sql из папки Others.
4. Прописать параметры базы данных из папки Server в файле config.js, порт сервера, на котором будет располагаться серверная часть, а так же уровень отладки.
5. Заменить адрес сайта "localhost" и порт "8181" на адрес, на котором будет располагаться серверная часть из папки Client в файле config.js.

Основной серверный файл для запуска - server.js;
Запуск через NodeJS - node server.js;
Запуск через демон (forever) NodeJS - forever start server.js;
