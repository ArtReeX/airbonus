1. Залить клиентскую часть из папки Client на сервер.
2. Залить серверную часть из папки Server на сервер.
3. Залить базу данных pilot.sql из папки Others.
4. Прописать параметры базы данных из папки Server в файле config.js, а так же порт сервера, на котором будет располагаться серверная часть.
5. Заменить адрес сайта akite.org:8181 на адрес, на котором будет располагаться серверная часть из папки Client в файлах index.html, 6.html, 7.html, 9.html. (Для этого можно сделать поиск с автозаменой слова в файлах.)

Основной серверный файл для запуска - main.js;
Запуск через NodeJS - node main.js;
Запуск через демон (Forever) NodeJS - forever start main.js;
